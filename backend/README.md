## Google Marker

### Prequisites
- [PYTHON 3](https://www.python.org/downloads/release/python-374/)

### Installing
1. Clone the project.

2. Install the following modules with pip.
    - flask_compress
    - flask_cors

3. In your terminal, run `python api.py`, this will start the local server on "http://127.0.0.1:8090/".
